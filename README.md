# Hi Series Official

Hi Series 官网，使用 Astro 构建，独立于现有项目。

## 页面结构

- 首页首屏：品牌主张、产品概览、主 CTA
- 产品陈列：HI API、Hi AI Image Studio
- 产品理念：清晰入口、低摩擦使用、持续演进
- 关于作者：独立开发者介绍
- 路线图：后续产品模块占位

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 目录

- `src/pages`：页面入口
- `src/components`：页面区块组件
- `src/data`：产品、原则、路线图等可维护数据
- `src/styles`：全局样式
- `public`：静态资源
